# MarketplaceParsers

## Installation

### Windows

```bash
git clone https://github.com/KsQRooW/marketplace_parsers.git
cd marketplace_parsers
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r .\requirements.txt
```

### Linux

```bash
git clone https://github.com/KsQRooW/marketplace_parsers.git
cd marketplace_parsers
python -m venv venv
source venv/bin/activate
pip install -r .\requirements.txt
```
